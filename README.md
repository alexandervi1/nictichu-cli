# NictichuCLI

**Agente de código multi-modelo conMCPs y Gemma** 🎨

Paleta de colores: **Cian/Turquesa (#06B6D4)**

## Instalación Rápida

### Instalar
```bash
pip install -e .
```

### Usar
```bash
nictichu interactive --model gemma:7b
```

## Características

- ✅ Multi-modelo: Ollama, Google AI, Vertex AI
- ✅ MCPs: Filesystem, Shell, Memory, Search
- ✅ CLI Interactivo con Rich
- ✅ Herramientas: Editor, Reviewer, Tester, Docs
